# Finance Dashboard UI

A modern, responsive finance dashboard built with React, Vite, and Bootstrap. Track your finances, visualize spending patterns, and manage transactions with role-based access control.

## Features

### Core Features
- 📊 **Dashboard Overview**: Summary cards with total balance, income, and expenses
- 📈 **Balance Trend**: Time-based visualization (daily/weekly/monthly)
- 🥧 **Spending Breakdown**: Categorical visualization of expenses
- 📝 **Transaction Management**: Add, edit, delete, and filter transactions
- 🔍 **Search & Filter**: Find transactions by description or category
- 👥 **Role-Based UI**: Simulated Admin/Viewer roles with different permissions
- 💡 **Financial Insights**: Smart observations about spending patterns
- 📱 **Fully Responsive**: Works on all screen sizes

### Technical Features
- React 18 with Hooks
- Vite for fast development
- Bootstrap 5 for styling
- Chart.js for beautiful visualizations
- Axios for API calls
- React Router for navigation
- Context API for state management
- Local storage for persistence

## Prerequisites

- Node.js (v14 or higher)
- npm or yarn
- Backend server running (see backend setup)
